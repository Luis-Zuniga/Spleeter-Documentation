Steps followed by the Spleeter Training

From command line, the Spleeter train command is input: spleeter train -p configs/musdb_config.json -d /path/to/musdb

The function 'main', from spleeter/_main_.py, is called.
After parsing the command, it checks to see whether it is a separate, train, or evaluate command. In this case, it is a train command.
The function 'entrypoint', from spleeter/commands/train.py, is imported.
The function 'load_configuration', from spleeter/utils/configuration.py, is called with input 'arguments.configuration'.
The function 'entrypoint', from spleeter/commands/train.py is called with inputs 'arguments' and 'params'.

From spleeter/audio/adapter.py, the get_audio_adapter function is called, takes input 'arguments.audio_adapter' and is attached to variable 'audio adapter'.
Loads the AVCon or FFMPEG audio adapter.

The function '_create_estimator', from spleeter/commands/train.py, is called with input 'arguments.audio_adapter'.
